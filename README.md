# Kitty domain

> a simple fast and efficient subdomain miner  


# Quick start

**build** 
```golang
$ go build .
```

**command parameter** 
>  -d string  
>    	target domain  
>  -dns string  
>    	target domain (default "8.8.8.8")  
>  -f string  
>    	set subdomain dict file (default "subdomains.txt")  
>  -o string  
>    	set out put file (default "result.txt")
